# Tech Stack

## Frontend (React + Vite)

- **React** – Component-based frontend framework
- **Vite** – Fast build tool and dev server
- **Axios** – Handles HTTP requests to the backend
- **Tailwind CSS** – Utility-first CSS framework for styling
- **React Router** – Client-side routing
- **React Hot Toast** – Simple toast notifications
- **Framer Motion** – Animation library for smooth UI transitions
- **HeroUI** – Ready-to-use UI components
- **Lucide React** - Icon library for react

## Backend (Spring Boot)

- **Spring Web (spring-boot-starter-webmvc)** – REST API support and MVC framework
- **Spring Data JPA (spring-boot-starter-data-jpa)** – Database access using JPA/Hibernate
- **H2 Database (runtime)** – In-memory database for development/testing
- **Spring Boot DevTools (runtime, optional)** – Live reload and faster development
- **Spring Boot Actuator (spring-boot-starter-actuator)** – Monitoring and management endpoints
- **Spring Boot Test Packages**
  - **spring-boot-starter-data-jpa-test** – Utilities for testing JPA repositories
  - **spring-boot-starter-webmvc-test** – Utilities for testing controllers

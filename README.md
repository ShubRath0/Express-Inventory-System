# Express Inventory System

🔗 Live Demo: [https://express-inventory-system-1.onrender.com/](https://express-inventory-system-1.onrender.com/)

## Tech Stack

### Frontend (React + Vite)

- **React** – Component-based frontend framework
- **Vite** – Fast build tool and dev server
- **Axios** – Handles HTTP requests to the backend
- **Tailwind CSS** – Utility-first CSS framework for styling
- **React Router** – Client-side routing
- **React Hot Toast** – Simple toast notifications
- **Framer Motion** – Animation library for smooth UI transitions
- **HeroUI** – Ready-to-use UI components
- **Lucide React** – Icon library for React

### Backend (Spring Boot)

- **Spring Web (spring-boot-starter-web)** – REST API support and MVC framework
- **Spring Data JPA (spring-boot-starter-data-jpa)** – Database access using JPA/Hibernate
- **Spring Boot DevTools** – Live reload and faster development (optional runtime)
- **Spring Boot Actuator (spring-boot-starter-actuator)** – Monitoring and management endpoints
- **Spring Validation (spring-boot-starter-validation)** – Input validation for APIs
- **Spring Security (spring-boot-starter-security)** – Authentication & authorization
- **PostgreSQL Driver** – Database connection
- **Lombok** – Reduces boilerplate code
- **JSON Web Tokens (jjwt-api, jjwt-impl, jjwt-jackson)** – Token-based authentication
- **SpringDoc OpenAPI** – API documentation UI
